Blender AI 模型拆件與 Boolean 實戰指南

針對 Tripo.ai 等 AI 生成模型，解決 Mesh 密度極唔平均、內部重疊、冇卡榫結構嘅終極互動教學。

🔧 核心概念：Modifier 與 Boolean

本區塊為您解析 Blender 中最重要嘅非破壞性編輯工具。Modifier（修改器）允許您隨時反悔或調整，而 Boolean（布林運算）則係製作卡榫嘅核心數學運算。透過下方嘅互動模擬器，您可以直觀理解三種 Boolean 嘅運作原理。

互動 Boolean 模擬器

物體 A (藍) 減去 物體 B (橙) -> 挖卡榫凹槽

Modifier (修改器)

Blender 嘅非破壞性編輯工具。加咗 Modifier 去切開一件物體，只要未禁「Apply」，隨時可以反悔或者調整位置。

Difference (差集) ⭐️ 最常用

物體 A 減去 物體 B。喺拆件流程中，主要用嚟喺主體上挖出完美脗合嘅卡榫凹槽 (母榫)。

Union & Intersect

Union 將 A 同 B 融為一體 (AI 爛面易 error，建議用 Voxel Remesh 代替)。Intersect 只保留重疊部分，可用於特定切件需求。

🚀 實戰拆件流程：製作完美卡榫

跟隨以下四個步驟，將已經用 P (Separate) 分開嘅模型部件（例如「身體」與「右臂」），製作出可供 3D 打印精準組裝嘅公母卡榫。點擊下方步驟進行切換。

第一步

召喚 Boolean 修改器

  • 1. 喺 3D 視圖，Left Click 選取你想挖窿嗰件主體 (例如「身體」)。
  • 2. 望向畫面右下角嘅 Properties Panel (屬性面板)。
  • 3. 搵一個 藍色士巴拿 (扳手) 圖示 🔧。呢個就係 Modifier Properties。
  • 4. 撳 Add Modifier -> 喺 Generate 嗰欄搵 Boolean。
第二步

建立卡榫柱體 (公榫)

  • 1. 撳 Shift + A -> Mesh -> Cylinder (圓柱體)。呢個會用嚟做卡榫。
  • 2. 用 S (Scale) 將圓柱體縮細到啱比例。
  • 3. 用 G (Grab/Move) 同 R (Rotate) 將圓柱體移去右臂嘅斷口位。
  • 💡 重要提示： 確保圓柱體有一半插喺身體入面，一半留喺外面。
第三步

執行 Boolean Difference (挖母榫)

  • 1. 重新選取「身體」(啱啱加咗 Boolean Modifier 嗰件)。
  • 2. 喺 Modifier 面板，確保 Operation 係 Difference。
  • 3. 喺 Object 旁邊，撳一下滴管圖示，然後去畫面點擊頭先整嘅圓柱體。
  • 4. 呢個時候可能見唔到變化，因為圓柱體仲喺度擋住個窿。
  • 5. 喺右上角 Outliner (大綱視窗)，搵到 Cylinder，點擊旁邊個眼睛圖示 (隱藏它)。成功就會見到圓形凹槽！
第四步

合併公榫到手臂

  • 1. 將隱藏咗嘅圓柱體顯示返出嚟。
  • 2. 選取圓柱體，然後 Shift + Left Click 選取「右臂」。(右臂會變成淺橙色，代表 Active Object)。
  • 3. 撳 Ctrl + J (Join)。
  • 🎉 完成！ 而家圓柱體同右臂變成咗同一件 Object。公母榫製作完畢，可以準備處理公差。

🔥 必做前置作業：Voxel Remesh (大掃除)

去右下角綠色三角形 Tab (Data Properties) 搵 Remesh。揀 Voxel，將 Voxel Size 設定為 0.01m，撳 Voxel Remesh。呢步會將所有重疊、爛面強制融合成一個實心物體。

📏 打印公差 (Tolerance) 處理

如果柱體同凹槽係 1:1 大細，3D 打印完係絕對插唔入嘅。必須預留 0.1mm - 0.2mm 嘅空隙。

  1. 喺完成挖窿 (Step 3) 之後。
  2. 選取你條柱體 (公榫)。
  3. 撳 Alt + S (Shrink/Fatten)。
  4. 向入縮極少極少 (大約 0.1mm - 0.2mm)。

⚠️ 快速除錯 (Troubleshooting)

常見 Boolean 錯誤及解決方案快速檢索矩陣。

問題：窿唔見咗 / 模型消失咗？

原因： 法線反轉 (Normals Flipped) 或有爛面。
解法： Tab 入 Edit Mode -> A 全選 -> Shift + N (Recalculate Normals)。唔得就要做 Voxel Remesh。

問題：切口邊緣好狗牙 (起格)？

原因： 模型本身面數 (Resolution) 唔夠高。
解法： Boolean 之前，加一個 Subdivision Surface Modifier 令佢滑啲，或者 Remesh 時較細個數值。
